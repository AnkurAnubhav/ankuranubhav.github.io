CSS Selectors

Function

HTML Tags

Tags

Function

media

to add a vidio, audio, image. Need to use src

figure and figcaption

The figure element is used to encapsulate media such as an image, diagram. The figcaption element is used to describe the media encapsulated within

section

defines elements in a document, such as chapters, headings, or any other area of the document with the same theme

aside

element is used to mark additional information that can enhance another elemen

article

holds content that makes sense on its own such as articles, blogs, and comments
